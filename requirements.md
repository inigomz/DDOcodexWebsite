# Requirements

## Runtime

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (bundled with Node.js)

## Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| [axios](https://www.npmjs.com/package/axios) | 1.16.0 | HTTP client for fetching DDO Wiki pages |
| [cheerio](https://www.npmjs.com/package/cheerio) | 1.2.0 | HTML parser for scraping item data from wiki tables |

## Installation

```bash
npm install
```

This will install all dependencies listed in `package.json`.
